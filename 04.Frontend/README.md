### eslint 설정 (VSCode)

- Extensions (Ctrl + Shift + X) -> eslint, prettier 설치<br><br>
  ![image](https://user-images.githubusercontent.com/44829661/193712256-31f51c01-ce97-4921-bd66-f04380f05667.png)<br><br>
  
- File - Preferences - Settings (Ctrl + ,) -> 오른쪽 상단의 Open Settings (JSON) 클릭 -> 하단의 내용 입력 후 Save (Ctrl + S)<br><br>
  ![image](https://user-images.githubusercontent.com/44829661/193712490-dd1e09a6-2183-45e5-b127-15accc65b5fb.png)

```javascript
  {
    "editor.formatOnSave": true,
    "editor.formatOnType": true,
    "editor.codeActionsOnSave": {
        "source.fixAll": true
    }
  }
```
